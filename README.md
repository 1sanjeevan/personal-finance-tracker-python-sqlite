# Personal Finance Tracker (Python + SQLite)

## 📌 Project Overview
This is a CLI-based Personal Finance Tracker developed using Python and SQLite to record income and expenses and calculate the available balance.

## 🚀 Features
- Add income
- Add expense
- View current balance
- Menu-driven command line interface
- Automatic database creation

## 🛠️ Technologies Used
- Python
- SQLite
- SQL

## ▶️ How to Run
1. Open terminal / command prompt
2. Navigate to the project folder
3. Run the command:

   python finance_tracker.py

## 📂 Database
The application automatically creates:
finance.db
